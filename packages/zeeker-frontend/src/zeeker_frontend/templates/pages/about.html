About Zeeker

Making data searchable and accessible to researchers and developers.

{# What This Is ----------------------------------------------- #}

What This Is

Zeeker is a searchable data catalogue. It provides structured access to curated datasets through both a web interface and a REST API. Each record carries a summary, identifying data, and a link to the original publication — full documents live at, and are read from, their original sources.

Built for researchers, developers, and anyone who needs to search through data efficiently. The catalogue is available through both the web interface and programmatic API access.

{# What You Can Do -------------------------------------------- #}

What You Can Do

Full-Text Search

Search across all available data with filtering and faceted navigation.

Source-Linked Records

Every catalogue record links back to the original publication for the full text.

Export Data

Download catalogue records as JSON or CSV for your own analysis.

REST API

Programmatic access to all data for building tools or conducting research.

{# How It Works ----------------------------------------------- #}

How It Works

Built on Datasette, an open-source tool for exploring and publishing data. Data is stored in SQLite databases and automatically updated from various sources.

{# Who Uses This ---------------------------------------------- #}

Who Uses This

Researchers

  • Track developments and trends
  • Search for references and precedents
  • Analyze patterns over time
  • Export data for further analysis

Developers

  • Build data-driven applications
  • Create monitoring and alert systems
  • Integrate data into other tools
  • Develop research and analysis workflows
