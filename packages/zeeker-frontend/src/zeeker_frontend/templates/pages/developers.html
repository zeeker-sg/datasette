Access catalogue data programmatically via the JSON API and CSV exports.

{# 1. API overview --------------------------------------------- #}

API overview

Every database and table on data.zeeker.sg is available through a REST API. You can retrieve table rows as JSON or CSV, filter and sort them, and perform full-text search — all without authentication. CORS is enabled on all endpoints.

The API is powered by Datasette. For full documentation, see the Datasette JSON API docs.

A machine-readable description is available at /llms.txt.

{# 2. Content policy ------------------------------------------- #}

Content policy

This site is a catalogue. Records carry summaries, identifying data (titles, citations, dates, courts, categories), and a link to the original publication. Full text columns are indexed for search relevance but are not distributed via the website, the JSON API, or CSV exports. To read a full document, follow the record's source_url.

{# 3. Stable URL patterns -------------------------------------- #}

Stable URL patterns

PatternDescription
/{database}/{table}.jsonRetrieve table rows as JSON
/{database}/{table}.csvExport table rows as CSV (tables without protected full-text columns; protected tables are JSON-only)
/{database}/{table}/{pk}.jsonRetrieve a single record as JSON
/search?q=...Full-text search across all databases (HTML); /-/search.json?q=... for JSON
{# 4. Common parameters ---------------------------------------- #}

Common parameters

ParameterDescription
_sizeNumber of rows to return (default 100, max 1000)
_nextPagination token from previous response
_shapeResponse shape: objects (default), arrays, array, object
_sortSort results by column (ascending)
_sort_descSort results by column (descending)
_searchFull-text search within a table (tables with a search index)
{column}__exactFilter rows by column value (also __contains, __gte, __lte, …)
{# 5. curl examples (only when databases non-empty) ------------ #} {% if databases %} {% set example_db = databases[0] %} {% set example_table = example_db.tables[0].name if example_db.tables else 'table' %}

curl examples

Try these commands in your terminal. Replace database and table names as needed.

List rows as JSON

curl "https://data.zeeker.sg/{{ example_db.name }}/{{ example_table }}.json?_size=10"

Download as CSV

curl "https://data.zeeker.sg/{{ example_db.name }}/{{ example_table }}.csv" -o data.csv

Search within a table

curl "https://data.zeeker.sg/{{ example_db.name }}/{{ example_table }}.json?_search=contract&_size=10"

Full-text search across all databases

curl "https://data.zeeker.sg/-/search.json?q=contract"
{% endif %} {# 6. Export formats ------------------------------------------- #}

Export formats

JSON

The default format. Use the _shape parameter to control structure:

  • objects — array of objects with column names as keys (default)
  • arrays — array of arrays, with a separate columns list
  • array — flat array of objects (no pagination metadata)
  • object — object keyed by primary key

CSV

Append .csv to a table URL. Use ?_stream=on for large exports that stream all rows without pagination. CSV export is available for tables without protected full-text columns; tables that carry protected columns are JSON-only.
