|----------|-------------|
| developer shell → Caddyfile on disk | The mutation point — file is read by Caddy at next start/reload |

| Caddy → datasette upstream | Existing trust boundary; @datasette matcher decides whether to traverse it |
| Caddy → frontend upstream | NEW trust boundary established by this plan; catch-all forwards untrusted Host headers and request bodies to frontend:8000 |
