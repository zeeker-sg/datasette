# Sketch Manifest

## Design Direction
The current data.zeeker.sg UI reads as "white and plain, no design" — the palette (deep petrol `#0A4F55`) and typography (Fraunces serif) are defined but barely spent. Sketches explore a more color-confident, editorially-weighted aesthetic — something that signals "civic data platform" with craft, not a default template. Three candidate palettes to compare: **paper + petrol** (warm off-white, quiet), **broadsheet** (cream + ink-red, editorial), **petrol-ink** (dark mode).

## Reference Points
- Editorial/data-journalism: The Pudding, ProPublica Data Store, Our World in Data
- Civic-data tone: UK Government Digital Service (but warmer), data.gov.sg (but with more craft)
- Typography direction: Fraunces italic displays, Inter body, JetBrains Mono for data/code
- User prompt: "very white and plain, no design… add more design signal"
