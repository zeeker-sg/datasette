Developer Portal

Access data programmatically via REST API, CSV exports, and SQL queries

API Overview

Every database and table on data.zeeker.sg is available through a REST API. You can retrieve data as JSON or CSV, run SQL queries, and perform full-text search -- all without authentication. CORS is enabled on all endpoints.

The API is powered by Datasette. For full documentation, see the Datasette JSON API docs.

A machine-readable description is available at /llms.txt.

Stable URL Patterns

Pattern Description
/{database}/{table}.json Retrieve table rows as JSON
/{database}/{table}.csv Export table rows as CSV
/{database}.json?sql=... Execute a SQL query and return results as JSON
/-/search.json?q=... Full-text search across all databases

Common Parameters

Parameter Description
_size Number of rows to return (default 100, max 1000)
_next Pagination token from previous response
_shape Response shape: objects (default), arrays, array, object
_sort Sort results by column (ascending)
_sort_desc Sort results by column (descending)

curl Examples

Try these commands in your terminal. Replace database and table names as needed.

List rows as JSON

curl "https://data.zeeker.sg/{{ example_db.name }}/{{ example_table }}.json?_size=10"

Download as CSV

curl "https://data.zeeker.sg/{{ example_db.name }}/{{ example_table }}.csv" -o data.csv

Run a SQL query

curl "https://data.zeeker.sg/{{ example_db.name }}.json?sql=SELECT+*+FROM+[{{ example_table }}]+LIMIT+10"

Full-text search

curl "https://data.zeeker.sg/-/search.json?q=contract"

Export Formats

JSON

The default format. Use the _shape parameter to control the response structure:

  • objects -- array of objects with column names as keys (default)
  • arrays -- array of arrays, with a separate columns list
  • array -- flat array of objects (no pagination metadata)
  • object -- object keyed by primary key

CSV

Append .csv to any table URL. Use ?_stream=on for large exports that stream all rows without pagination.

SQLite Database Download

Download the entire database file directly for offline analysis. Available from each database page via the download link.
