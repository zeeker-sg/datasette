Method: SQL query to count mentions by month/year

       COUNT(*) as mentions
FROM your_table

SQL for Researchers

SQL might look intimidating, but it is just a way to ask specific questions about data. Here are the most useful patterns:

Find Recent Records

SELECT title, date
FROM your_table
WHERE date > '2024-01-01'
ORDER BY date DESC
LIMIT 20

Gets the 20 most recent records since January 2024

Search by Keyword

SELECT title, date
FROM your_table
WHERE LOWER(text) LIKE '%keyword%'
   OR LOWER(title) LIKE '%keyword%'
ORDER BY date DESC

Finds all records mentioning "keyword" anywhere in title or text

Mentions Over Time

SELECT strftime('%Y-%m', date) as month,
       COUNT(*) as mentions
FROM your_table
WHERE LOWER(text) LIKE '%keyword%'
GROUP BY month
HAVING month IS NOT NULL
ORDER BY month DESC
LIMIT 24

Shows keyword-related records by month for the last 2 years

Complex Multi-term Search

SELECT title, date
FROM your_table
WHERE (LOWER(text) LIKE '%term one%'
   OR LOWER(text) LIKE '%term two%'
   OR LOWER(text) LIKE '%term three%')
   AND date > '2023-01-01'
ORDER BY date DESC

Records matching multiple terms from 2023 onwards

Weekly Activity Trends

SELECT
  strftime('%Y-W%W', date) as week,
  COUNT(*) as record_count
FROM
  your_table
WHERE
  date >= '2024-01-01'
  AND date IS NOT NULL
GROUP BY
  strftime('%Y-W%W', date)
ORDER BY
  strftime('%Y-W%W', date) DESC
LIMIT
  12

Weekly record counts for the last 12 weeks

SQL Tips

  • Use % as wildcard: '%contract%' finds "contract", "contracts", "contractual"
  • Combine searches: OR for either term, AND for both terms
  • Sort results: ORDER BY date DESC for newest first
  • Limit results: LIMIT 50 to avoid overwhelming output
  • Date ranges: date BETWEEN '2023-01-01' AND '2023-12-31'

Getting Data Out

Once you have found what you need, here is how to export it for further analysis:

For Excel/Google Sheets

Add .csv to any URL. CSV files open directly in spreadsheet software.

/-/search.csv?q=your+search
/{database}/{table}.csv?_size=1000

For Programming (Python/R/etc)

Add .json to any URL. JSON is easier to work with in code.

import requests
url = 'https://data.zeeker.sg/{database}/{table}.json'
data = requests.get(url).json()

Entire Database

Download the full SQLite database file for offline analysis.

/{database}.db
# Opens in DB Browser for SQLite, etc.
