SQL Reference

Common Patterns

  • Text search: WHERE LOWER(column) LIKE '%term%'
  • Date range: WHERE date >= '2024-01-01'
  • Multiple conditions: WHERE condition1 AND condition2
  • Null checks: WHERE column IS NOT NULL
  • Limit results: ORDER BY column DESC LIMIT 50

Useful SQLite Functions

  • strftime('%Y', date) - Extract year from date
  • LENGTH(text) - Get text length
  • COUNT(*) - Count rows
  • GROUP BY column - Group results
  • DISTINCT column - Get unique values

Tips

  • Table names with spaces: Use double quotes like "table name"
  • Case-insensitive search: Use LOWER(column) for text comparisons
  • Performance: Always use LIMIT for large tables
